<h2>社内における社員と商品の管理システム</h2>

概要<br>
本アプリケーションでは、社内における社員情報の管理、出退勤登録、商品登録、商品レビューを一元管理することができます。

主な機能<br>
・社員登録<br>
・所属や肩書による閲覧権限の付与<br>
・出退勤時間の打刻<br>
・出退勤時刻の編集機能<br>
・商品情報登録<br>
・商品情報ソート機能<br>
・商品レビュー<br>

開発環境<br>
PHP 8.4<br>
MariaDB 10.4.32<br>
Laravel 10.0<br>
